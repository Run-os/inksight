# 图片资源目录

设备图片页从这里读取图片。由**配套生成脚本**（样式规则后续补充）把图片处理成适合
4.2" 反射屏（400×300、1bpp 黑白）的格式后放入本目录。

## 约定（本期规划，后续脚本实现）

- 支持扩展名：`.bmp` / `.png` / `.jpg` / `.jpeg` / `.gif`
- 文件名按字母序决定展示顺序（即 `slot` 顺序）；例如 `01.bmp` / `02.bmp` ...
- 建议尺寸：内容区 400×214（状态栏 52px + 底栏 18px 由设备原生绘制）
- 推荐 1-bit 单色（黑白），保证反射屏可读性

## 设备接口

- `GET /api/images/manifest` → 返回 `{version, count, images:[{slot,name,url}]}`
  - 设备比较本地缓存的 `version`，不变则跳过下载
- `GET /api/images/{name}` → 返回图片二进制

> 本目录不纳入版本库（见 .gitignore）；部署时由生成脚本写入真实图片。

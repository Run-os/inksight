INKSIGHT

— 诸事有序 · 设备后端服务

设备接口

  • GET /api/todos — 待办列表（默认显示页，对接滴答清单）
  • GET /api/images/manifest — 图片清单（按目录顺序）
  • GET /api/images/{name} — 单张图片二进制

管理 / 授权

  • GET /admin/dida365/auth — 开始滴答清单 OAuth 授权
  • GET /health — 健康检查
▶ 连接滴答清单

首次部署需在此完成授权，后端会把令牌保存到 .dida_tokens.json。

说明

设备开机默认进入待办页，通过物理按键切换到图片页。配置项见站点根目录 .env。 图片资源放置于 images/ 目录，由配套生成脚本产出。
